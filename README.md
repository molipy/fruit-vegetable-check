# fruit-vegetable-check
基于深度学习的【果蔬识别】系统~Python+2026原创+计算机毕设

## 一、介绍
本系统面向果蔬图像智能识别场景进行设计与实现，围绕“上传图片、自动识别、结果展示、历史留存、信息管理”构建了一套完整的 Web 应用。系统可识别的类别包括土豆、圣女果、大白菜、大葱、梨、胡萝卜、芒果、苹果、西红柿、韭菜、香蕉、黄瓜共 12 类常见果蔬。前端采用 Vue3 与 Element Plus 搭建交互界面，后端基于 Flask 框架实现接口服务，并结合 TensorFlow 与 ResNet50 模型完成核心识别任务。用户登录后可上传果蔬图片，系统将自动完成图片预处理、模型推理、结果排序与历史记录存储。
